File System Access API

The File System Access API offers new features to manipulate local files providing an improved experience to our users like a native app. We don't have to download the file each time we want to edit it, but we can save it directly. We can even implement an auto-save functionality.

This feature is available only over secure connections (HTTPS) and through user interaction.
Active by default from Chrome 86. In older Chrome versions you might need to enable the related flag in Chrome to test it (chrome://flags/#native-file-system-api).
