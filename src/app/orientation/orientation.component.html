DeviceOrientationEvent API

The DeviceOrientationEvent API provides web developers events carrying the information about the physical orientation of the target device where the web app is running.

This feature works only in secure contexts (HTTPS). Use a mobile device for best results.

Toggle Card Details

Tilt the device towards you and back to toggle the card details.
